# IRC bot for school project

This project uses [otitcode-wiki](https://github.com/egetin/otitcode-wiki) api to manage data and authorization.

### Usege
```shell
go get github.com/heppu/pwp-bot
go install github.com/heppu/pwp-bot
pwp-bot
```
